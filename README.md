## About Me

I am a Cloud and DevOps Engineer focused on designing, automating, and operating reliable infrastructure across **Microsoft Azure** and **Amazon Web Services**.

My work centers on Infrastructure as Code, CI/CD, container orchestration, cloud security, monitoring, and operational efficiency. I enjoy turning manual processes into repeatable automation and helping teams deliver software with greater speed and confidence.

- Designing scalable and resilient cloud architectures
- Provisioning infrastructure with Terraform and configuration automation
- Building CI/CD pipelines for reliable software delivery
- Containerizing workloads with Docker and orchestrating them with Kubernetes
- Improving observability with cloud-native monitoring and logging
- Automating operational tasks with Python, Bash, and PowerShell
